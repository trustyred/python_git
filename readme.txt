#python test day 2
  今天主要复习了关于文件操纵，字符串操作，列表。
打开文件可以使用open方法（内建方法）或者file类
两种形式基本一样，都会返回一个file对象
打开有几种mode，r,w,a，分别是读，写，追加
还有rw,wr等，要及其注意wr，是会一开始就清空文件的。很危险。
但是rw不会，但是如果写入数据的时候会对原文件的数据进行覆盖。
  关于字符串的操作有几个是以前没有很注意的，比如strip()，split()
join()。strip()是用来清除开头和结尾的字符什么都不添加的时候是清除
空格,换行等符号。split()和awk的一些功能是一样的，只不过更简单，将
字符串根据相应的分割符分成列表。join是将列表转换成为字符串，
并且以你指定的分隔符分开。
